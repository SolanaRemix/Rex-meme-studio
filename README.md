# 🦖 REX Meme Studio  
**AI‑powered meme generator with Solana rewards, Frames, Blinks, and modular multi‑service architecture.**

REX Meme Studio is a production‑grade, open‑source meme engine built for creators, communities, and token ecosystems.  
It provides a clean UI, deterministic backend, and Solana‑native reward flows — all in a modular monorepo ready for scaling.

---

## 🚀 Features
- **AI Meme Generation** — deterministic, prompt‑driven meme creation  
- **Solana Rewards** — integrates with REX token flows  
- **Frames + Blinks** — Farcaster‑native distribution  
- **Modular Architecture** — UI, engine, rewards, and design system separated  
- **Operator‑Grade Tooling** — reproducible builds, clean scripts, zero drift  
- **Open‑Source Governance** — Apache 2.0 + CC BY‑NC‑ND licensing model  

---

## 📦 Tech Stack
| Layer | Technology |
|-------|------------|

| Backend | Node.js, modular services |
